\## Getting Started

To run this program:

1. Install MINGW32 Compiler
2. Install SDL3 Library
3. Open in Cursor or Visual Studio Code
4. Modify the includes in 'c\_cpp\_properties.json' and 'tasks.json' to point to the SDL3 library
5. Modify the libs section in 'tasks.json' to point to the SDL3 library
6. Run the program from the entry point on ArcadeApp.cpp

NOTE: The SDL3.dll may need to be placed in the project source folder: https://github.com/libsdl-org/SDL
